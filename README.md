## Project Overview

Mellow Brew is a **complete, production‑ready cafe website** designed for a modern coffee shop. It combines a classic, warm aesthetic with practical features: a secure login gateway, a two‑page menu system with item modals, a table reservation form, a full‑screen Bootstrap carousel, a photo gallery, and a contact section. The site is fully responsive and works seamlessly on desktops, tablets, and mobile devices.

## Key Features

| Feature               | Description                                                                 |
|-----------------------|-----------------------------------------------------------------------------|
| **Login Protection**  | Simple overlay login (any non‑empty credentials) to access the full site.  |
| **Hero Carousel**     | Bootstrap 5 carousel with high‑resolution Unsplash images & captions.      |
| **Two‑Page Menu**     | Tabs: *Classic Favorites* (4 items) and *Extra Delights* (5 items).        |
| **Item Modals**       | Click any menu card → modal shows full description, price, and image.      |
| **Table Booking**     | “Explore Cafe” button opens a reservation form (name, email, date, time, guests). |
| **Gallery**           | 6 realistic cafe photos in a responsive grid (hover zoom).                 |
| **About & Contact**   | Story, address, phone, email, opening hours, social media placeholder.     |
| **Logout**            | Navbar button to return to the login screen.                               |
| **Responsive Design** | Mobile‑friendly layout using Bootstrap 5 + custom CSS.                     |

##Tech Stack

- **HTML5** – semantic structure  
- **CSS3** – custom styling, Flexbox/Grid, animations  
- **JavaScript (ES6)** – dynamic menu rendering, modal handling, booking logic  
- **Bootstrap 5** – carousel, modals, grid system, navbar  
- **Font Awesome 6** – icons throughout  
- **Google Fonts** – Playfair Display (headings) + Inter (body)  
- **Unsplash** – realistic, high‑quality stock images (all externally hosted)

## How to Run Locally

1. Clone this repository or download the `index.html` file.  
2. Open `index.html` in any modern web browser (no build step required).  
3. Use any **non‑empty** User ID/Mobile and Password to enter the cafe.  

*No server or database needed – all data is front‑end demo.*
